🍔 Swiggy Data Dashboard (Power BI)
📌 Project Overview
This project is a Power BI dashboard built to analyze food delivery data inspired by Swiggy. It provides valuable insights into order trends, customer preferences, restaurant performance, and revenue patterns.
The dashboard transforms raw data into meaningful visualizations to help understand business performance in the online food delivery industry.

🚀 Features
📦 Total Orders & Revenue Analysis
🏙️ City-wise Performance Insights
🍽️ Restaurant-wise Sales Analysis
🌮 Popular Cuisines & Food Trends
📅 Daily & Monthly Order Trends
🎛️ Interactive Filters & Slicers

🔍 Key Features:
Total orders and revenue analysis
City-wise and restaurant-wise performance
Popular cuisines and food trends
Time-based order analysis (daily/monthly trends)
Interactive filters and slicers for dynamic exploration

🎯 Objective:
The main objective of this project is to demonstrate skills in:
Data visualization using Power BI
Data cleaning and transformation
Creating interactive dashboards
Extracting actionable insights from food delivery data
This project is ideal for showcasing data analytics and business intelligence skills in a real-world use case similar to food delivery platforms.

📊 Key Insights
Identifies top-performing restaurants and cities
Highlights most popular cuisines among customers
Shows order trends over time
Helps understand customer ordering behavior
